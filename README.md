ASTEROIDS

This is just a simple Asteroids project from Boot.dev.

The main point of this project was to showcase Python's use of methods, objects, inheritance, and libraries.

The player is able to rotate left/right, move forward/back, and shoot.
The asteroids spawn in 3 varying sizes from randomized screen edges.
When an asteroid is shot, it will split into multiples of the next smaller size (i.e. large --> medium --> small).
When a small asteroid is hit, it will disappear from the screen.
Finally, if an asteroid hits the player, the game will end.
